### Step 6: Update CI/CD

Add architecture validation to your CI/CD pipeline:

```yaml

  run: |
    python tools/validation/validate-architecture.py --strict

- name: Check Technical Debt
  run: |
    python tools/validation/check-technical-debt.py --threshold 0

  script:
    - python tools/validation/validate-architecture.py --strict
    - python tools/validation/check-technical-debt.py --threshold 0
```

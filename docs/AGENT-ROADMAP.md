## Contributing

To propose a new agent for the roadmap, create a GitHub issue with:
1. Agent name and proposed archetype (Reviewer, Architect, Domain Expert, Orchestrator, Enforcer)
2. Problem it solves that no existing agent covers
3. Target users and use cases

To build a roadmap agent, use the [Agent Creation Pipeline](AGENT-CREATION-GUIDE.md) with research prompts and reference archetypes.

# Programme SDLC Method

The programme method coordinates multiple delivery teams working toward a shared goal. It introduces formal waterfall phase gates for requirements, design, test, and code.

Each phase gate must be passed before the next phase begins, and a mandatory cross-phase review checks that downstream work traces back to upstream artefacts. Four phase-gate validators enforce the gates automatically.

The programme method is appropriate for large multi-team efforts that need explicit coordination and traceability. It is heavier than the single-team method and should not be used by a single team.
